# One80 English — logo concepts

Five distinct logo directions. The core idea behind the name is **180° = a complete turnaround / transformation in your English** — every concept expresses that, without the tired category clichés (no owl, globe, open book, grad cap, or speech bubble).

Type: **Poppins** (geometric sans) — light/medium "One" + bold "80" so the number that carries the meaning pops. Swap to a licensed face (e.g. Circular, Gotham, Geomanist) for final production if desired.

All marks are built to work in **one color** and to read at **favicon size**. View them together in [index.html](index.html).
